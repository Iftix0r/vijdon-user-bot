# Vijdon Taxi Bot - To'liq Funksiyalar

## Asosiy Funksiyalar

### 1. Telegram Akkauntga Kirish
- Telefon raqam orqali kirish
- SMS kod tasdiqlash
- 2FA parol qo'llab-quvvatlash
- Bir nechta akkaunt almashish

### 2. Guruhlarni Boshqarish
- Barcha guruhlarni avtomatik import qilish
- Guruhlarni faollashtirish/nofaol qilish
- Guruh statistikasi
- Guruhlarni o'chirish

### 3. AI Buyurtma Aniqlash
- Yo'lovchi/Haydovchi aniqlash
- Qayerdan/Qayerga ajratish
- Narx aniqlash
- Telefon raqam ajratish
- O'zbek tilida ishlash

### 4. Buyurtmalarni Qabul Qilish
- ✅ Qabul qilish tugmasi
- ❌ Rad etish tugmasi
- Haydovchi ma'lumotlari
- Mijoz bilan bog'lanish

### 5. Admin Panel
- Premium Dashboard (modern dizayn)
- Chart.js statistikalar (yo'lovchi vs haydovchi)
- So'nggi xabarlar ro'yxati
- Guruhlar ro'yxati va holati
- System monitoring (CPU, RAM)
- Blacklist boshqaruvi

### 6. Blacklist Tizimi
- User ID orqali bloklash
- Bloklangan foydalanuvchilar xabarlari tahlil qilinmaydi
- Admin panel orqali boshqarish

### 7. Sozlamalar
- Buyurtmalar guruhini o'zgartirish
- OpenAI API kalitini o'zgartirish
- Bot tokenini o'zgartirish

### 7. Xatolarni Boshqarish
- Barcha xatolar adminga yuboriladi
- Xato traceback
- Xato vaqti va ma'lumotlari

## Komandalar

### Admin Komandalar
- `/start` - Boshqaruv paneli
- `/add <chat_id> <nomi>` - Guruh qo'shish
- `/stats` - Statistika
- `/broadcast <xabar>` - Barcha guruhlarga xabar
- `/settarget <chat_id>` - Buyurtmalar guruhini o'zgartirish
- `/setapi <api_key>` - OpenAI API kalitini o'zgartirish
- `/restart` - Botni qayta ishga tushirish
- `/help` - Yordam
